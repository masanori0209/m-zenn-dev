# m-zenn-dev

## 概要
- Zennでの活動記録


## Zenn CLI コマンド

### 記事の作成
```bash
# 新規記事の作成
npx zenn new:article

# プレビュー表示
npx zenn preview
```

### 本の作成
```bash
# 新規本の作成
npx zenn new:book

# 本のチャプター作成
npx zenn new:book:chapter
```

### その他のコマンド
```bash
# ヘルプの表示
npx zenn --help

# バージョン確認
npx zenn --version
```
